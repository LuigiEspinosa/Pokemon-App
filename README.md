# Pokémon App

- Frontend: https://pokemon.cuatro.dev
- Backend: https://api.pokemon.cuatro.dev

## Structure

```txt
.
├─ apps/
│ ├─ frontend/            # Next.js 16 (App Router, RSC) + Tailwind
│ └─ backend/             # Express, caches + PokeAPI proxy
├─ docker-compose.yml     # Local dev
└─ README.md
```

## Local dev

Docker Required

```zsh
pnpm install-app
pnpm dev-app
```
